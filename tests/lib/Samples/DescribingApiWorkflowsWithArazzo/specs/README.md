# Describing API workflows with Arazzo samples

These files are copies of sample specs published in the `frankkilcommins/describing-api-workflows-with-arazzo` repository:

- Source repository: <https://github.com/frankkilcommins/describing-api-workflows-with-arazzo>
- Source directory: <https://github.com/frankkilcommins/describing-api-workflows-with-arazzo/tree/main/specs>

The copied set includes the non-empty Arazzo documents and their companion OpenAPI and JSON files used by those specs.

The upstream `specs/bnpl.arazzo.yaml` file is currently a zero-byte placeholder, so it is intentionally not included in the parser dataset.
