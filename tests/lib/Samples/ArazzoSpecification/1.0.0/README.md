# Arazzo Specification 1.0.0 examples

These files are copies of the upstream examples published in the OAI Arazzo Specification repository:

- Source repository: <https://github.com/OAI/Arazzo-Specification>
- Source directory: <https://github.com/OAI/Arazzo-Specification/tree/main/examples/1.0.0>

The copied set includes both the Arazzo example documents and the peer OpenAPI documents that live alongside them upstream, so referenced companion files stay available in the local test dataset.

They were copied into this repository to provide a stable parser test dataset for `BinkyLabs.OpenApi.Arazzo.Tests`.
